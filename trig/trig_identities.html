Trigonometric Identities

Definitions

$$\begin{align} \sin \theta &= x\text{-coordinate on unit circle} \\[.75em] \cos \theta &= y\text{-coordinate on unit circle} \\[.75em] \tan \theta &= \frac{\sin \theta}{\cos \theta} \\[.75em] \cot \theta &= \frac{\cos \theta}{\sin \theta} \\[.75em] \sec \theta &= \frac{1}{\cos \theta} \\[.75em] \csc \theta &= \frac{1}{\sin \theta} \end{align}$$

Reciprocal Identities

\begin{align} \sin \theta &= \frac{1}{\csc \theta} \\[.75em] \cos \theta &= \frac{1}{\sec \theta} \\[.75em] \tan \theta &= \frac{1}{\cot \theta} \\[.75em] \cot \theta &= \frac{1}{\tan \theta} \\[.75em] \sec \theta &= \frac{1}{\cos \theta} \\[.75em] \csc \theta &= \frac{1}{\sin \theta} \end{align}

Pythagorean Identities

\begin{equation} \sin^2 \theta + \cos^2 \theta = 1 \end{equation} \begin{equation} \tan^2 \theta + 1 = \sec^2 \theta \end{equation} \begin{equation} 1 + \cot^2 \theta = \csc^2 \theta \end{equation}

Sum/Difference Identities

\begin{align} \sin(\alpha \pm \beta) &= \sin \alpha \cos \beta \pm \cos \alpha \sin \beta \\[.75em] \cos(\alpha \pm \beta) &= \cos \alpha \cos \beta \mp \sin \alpha \sin \beta \\[.75em] \tan(\alpha \pm \beta) &= \frac{\tan \alpha \pm \tan \beta}{1 \mp \tan \alpha \tan \beta} \end{align}

Double Angle Identities

\begin{align} \sin 2\theta &= 2 \sin \theta \cos \theta \\[.75em] \cos 2 \theta & = \cos^2 \theta - \sin^2 \theta \\[.75em] & =2\cos^2 \theta - 1 \\[.75em] & =1-2\sin^2 \theta \\[.75em] \tan2 \theta & = \frac{2 \tan \theta}{1 - \tan^2 \theta} \end{align}

Half Angle Identities

\begin{align} \sin \frac{\theta}{2} &= \pm \sqrt{\frac{1-\cos \theta}{2}} \\[.75em] \cos \frac{\theta}{2} &= \pm \sqrt{\frac{1+\cos \theta}{2}} \\[.75em] \tan \frac{\theta}{2} &= \pm \sqrt{\frac{1 + \cos \theta}{1 - \cos \theta}} = \frac{\sin \theta}{1 + \cos \theta} = \frac{1 - \cos \theta}{\sin \theta} \end{align}

Odd & Even Identities

\begin{align} \sin (-\theta) &= -\sin \theta && \text{odd} \\[.75em] \cos(-\theta) &= \cos \theta && \text{even} \\[.75em] \tan(-\theta) &= -\tan \theta && \text{odd} \\[.75em] \cot(-\theta) &= -\cot \theta && \text{odd} \\[.75em] \sec(-\theta) &= \sec \theta && \text{even} \\[.75em] \csc(-\theta) &= -\csc \theta && \text{odd} \end{align}

Complement/Cofunction Identities

Note: the complement of an angle $\theta$ is the angle $\frac{\pi}{2} - \theta$. \begin{align} \sin \left(\frac\pi 2-\theta\right) &= \cos \theta \\[.75em] \cos \left(\frac\pi 2-\theta\right) &= \sin \theta \\[.75em] \tan \left(\frac\pi 2-\theta\right) &= \cot \theta \\[.75em] \cot \left(\frac\pi 2-\theta\right) &= \tan \theta \\[.75em] \sec \left(\frac\pi 2-\theta\right) &= \csc \theta \\[.75em] \csc \left(\frac\pi 2-\theta\right) &= \sec \theta \end{align}

Period Identities

\begin{align} \sin (\theta + 2\pi) &= \sin \theta && \text{period }2\pi \\[.75em] \cos (\theta + 2\pi) &= \cos \theta && \text{period }2\pi \\[.75em] \tan (\theta + \pi) &= \tan \theta && \text{period }\pi \\[.75em] \cot (\theta + \pi) &= \cot \theta && \text{period }\pi \\[.75em] \sec (\theta + 2\pi) &= \sec \theta && \text{period }2\pi \\[.75em] \csc (\theta + 2\pi) &= \csc \theta && \text{period }2\pi \end{align}

Half-Period Identities

\begin{align} \sin (\theta \pm \pi) &= -\sin \theta && \text{period }2\pi \\[.75em] \cos (\theta \pm \pi) &= -\cos \theta && \text{period }2\pi \\[.75em] \tan \left(\theta \pm \frac{\pi}{2}\right) &= -\cot \theta && \text{period }\pi \\[.75em] \cot \left(\theta \pm \frac{\pi}{2}\right) &= -\tan \theta && \text{period }\pi \\[.75em] \sec (\theta \pm \pi) &= -\sec \theta && \text{period }2\pi \\[.75em] \csc (\theta \pm \pi) &= -\csc \theta && \text{period }2\pi \end{align}

Inverse Function Cancellation Identities

\begin{align} \sin \left( \arcsin x \right) &= x && \text{only if } x\in [-1,1] \\[.75em] \cos \left( \arccos x \right) &= x && \text{only if } x\in [-1,1] \\[.75em] \tan \left( \arctan x \right) &= x && \\[.75em] \arcsin \left( \sin \theta \right) &= \theta && \text{only if } \theta \in \left[ -\frac{\pi}{2}, \frac{\pi}{2} \right] \\[.75em] \arccos \left( \cos \theta \right) &= \theta && \text{only if } \theta \in [0, \pi] \\[.75em] \arctan \left( \tan \theta \right) &= \theta && \text{only if } \theta \in \left( -\frac{\pi}{2}, \frac{\pi}{2} \right) \end{align}

Sum-to-Product Identities

\begin{align} \sin \alpha \pm \sin \beta &= 2 \sin \left( \frac{\alpha \pm \beta}{2} \right) \cos \left( \frac{\alpha \mp \beta}{2}\right) \\[.75em] \cos \alpha + \cos \beta &= 2 \cos \left( \frac{\alpha + \beta}{2} \right) \cos \left( \frac{\alpha - \beta}{2}\right) \\[.75em] \cos \alpha - \cos \beta &= -2 \sin \left( \frac{\alpha + \beta}{2} \right) \cos \left( \frac{\alpha - \beta}{2}\right) \end{align}

Law of Sines

\begin{equation} \frac{\sin A}{a} = \frac{\sin B}{b} = \frac{\sin C}{c} \end{equation}

Law of Cosines

\begin{align} a^2 &= b^2 + c^2 - 2bc \cos A \\[.75em] b^2 &= a^2 + c^2 - 2ac \cos B \\[.75em] c^2 &= a^2 + b^2 - 2ab \cos C \end{align}
